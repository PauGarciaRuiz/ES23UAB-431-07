# Requirement 
### ID:  
RF-1-05 
### Type/Category: 
Funcional
### Version:  
1.0 
### Title:  
Crear perfil per fer la primera comanda
### Description:  
El perfil de l'usuari haurà d'indicar la seva adreça de recollida i, opcionalment, l'usuari podrà indicar restriccions alimentàries (com al·lèrgies o menjars que no vol rebre). 
### Relations:  
[RF-1-02](RF-1-02.md) 
### Comments:  
Aquest requisit actua com a bloqueig previ per a qualsevol usuari nou que intenti fer el seu 
primer pagament o comanda. El nom d'usuari haurà de ser validat pel sistema perquè 
compleixi els requisits.
